## idea快捷键



| 快捷键             | 作用                               |
| ------------------ | ---------------------------------- |
| psvm               | 快速生成main方法                   |
| sout               | 快速生成打印语句                   |
| shift +enter       | 另起一行                           |
| ctrl + shift +↑/↓  | 代码移动                           |
| ctrl +y            | 删除代码                           |
| ctrl +/            | 单行注释                           |
| ctrl + shift +/    | 多行注释                           |
| ctrl +art +t       | 快速补全流程控制结构/异常处理/上锁 |
| ctrl +shift +enter | 换行并补全代码尾部符号             |
| alt +enter         | 补全变量赋值                       |
| ctrl +alt +space   | 显示联想提示                       |
| shift +shift       | 查找源码                           |
| ctrl +alt +l       | 格式化代码                         |
|                    |                                    |
